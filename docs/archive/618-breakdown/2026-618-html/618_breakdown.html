🎯 战场一：老客保回购（¥66.6万 Gap）

🎯 战场二：达播人群入召回池
目标：达播保回购 策略： • 达播当天：捞近半年达播人群-回购提醒 • 达播后7天：关联推荐产品
🎯 战场三：新客补充战
目标：淘客GSV 从 ¥485万 → ¥574万（拉平Y25） 策略： • 货架端：妆品小样，持续派样-核心追踪新客回购率 • 达播端：聚焦浏览未购人群-集中推送达播信息 • 公域：参与所有的能够参与的618派样活动+云台短信截流竞品人群-引流拉新
🎯 战场四：会员渗透战（56%入会率保卫）
