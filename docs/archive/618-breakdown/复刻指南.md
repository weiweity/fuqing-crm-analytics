## 复刻流程

### Step 1：准备数据 → 修改 `gen_大促拆解模板.py`

1. 打开 `gen_大促拆解模板.py`
2. 修改日期参数：
   ```python
   YEAR = 2026
   CAMPAIGN = "双十一"   # 或 "618"
   ```
3. 修改 SQL 查询日期范围（拉取对应期间数据）
4. 运行脚本生成新 Excel：
   ```bash
   cd "/Users/hutou/Desktop/fuqin-date"
   python "拆解/26年618拆解文档/26年618html生成的文档/gen_大促拆解模板.py"
   ```
5. 输出的 Excel 命名为 `26年双十一拆解_日期.xlsx`

---

### Step 2：生成 HTML 风格原型

1. 用 Excel 数据手动重建 HTML 布局（参考 `618_breakdown.html` 结构）
2. 关键样式参考 `618_breakdown.png` 截图
3. 或让 AI 照着 `618_breakdown.html` 的结构重建，替换数据

**核心板块（不可少）：**
- KPI 指标卡（GSV目标/老客/新客/会员）
- 核心结论（北极星指标 + 关键策略）
- 每日趋势图（GSV + 订单数）
- 月度汇总（订单+新客趋势）
- GSV 构成（饼图/柱状图）

---

### Step 3：创建飞书文档

1. 在飞书新建空白文档
2. 照着 HTML 板块顺序，依次手动创建对应内容块
3. **指标卡用多维表格**：创建 KPI 卡片表 → 字段：`指标名称`（文本）+ `数值_数字`（数字）
4. **仪表盘**：基于多维表格创建仪表盘，参考 `618_breakdown_feishu.md` 的组件配置
